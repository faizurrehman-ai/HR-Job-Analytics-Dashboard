# 🚀 HR Job Change Prediction (Machine Learning)

Built a machine learning model to predict whether an employee is likely to leave or stay based on key factors such as experience, education, training hours, and city development index.

## ⚡ Key Features
- Performed data cleaning, missing value handling, and outlier detection  
- Conducted exploratory data analysis with visual insights  
- Applied feature engineering and encoding techniques  
- Trained a Random Forest model for classification  

## 📊 Impact
- Successfully predicted employee job change behavior with strong classification performance  
- Identified key factors influencing employee retention and turnover  

## 🛠️ Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn  

## 📦 Output
- Trained ML model saved as `hr_model.pkl` for deployment or reuse
